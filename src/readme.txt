The Nosnitor default theme provides flexable yet sleek theme for your WordPress site. 

== Description ==
The Nosnitor default theme is designed to provide elegant yet consistent formatting, in a feature rich theme. Nearly everything can be easily modified to accommodate many different purposes. Nosnitor is designed to be adaptable to a wide range of websites, whether you’re running a photo blog, launching a new business, or supporting a non-profit. This theme is Responsive, and should display well on a variety of devices and window or screen sizes.
